<a name="背景介绍"></a>  
## 背景介绍  
在移动端设备上，“手势密码”成为一个很常用的UI组件 <br> 
用户用手指按顺序依次划过9个原点中的若干个（>=5），如果划过的点的数量和顺序与之前用户设置的相同，那么当用户的手指离开屏幕的时候，判断为密码输入正确，否则密码错误。

<a name="项目介绍"></a>  
## 项目介绍  
手势密码组件，允许用户设置手势密码和验证手势密码
